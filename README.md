# Data Center Asset Tracker

## Project Overview
This Python-based utility is designed for enterprise infrastructure management. It automates the cleaning and validation of server asset data, ensuring that data center inventories remain accurate and actionable.

## Key Features
* **IP Validation:** Uses the `ipaddress` library to validate IPv4/IPv6 addresses and normalize formatting.
* **Status Monitoring:** Automatically flags servers with 'Offline' or 'Critical' statuses for immediate attention.
* **Data Transformation:** Converts messy, unformatted CSV data into a structured, searchable JSON database.
* **Logging:** Implements standard Python logging for enterprise-grade error tracking.

## Technical Skills Demonstrated
* Data Cleaning & Normalization
* Python Standard Library (`json`, `csv`, `ipaddress`, `logging`)
